# Data Science Training

A structured training repository covering Data Science, Machine Learning, Deep Learning, and AI/ML topics organized into weekly modules with scenario-based exercises and capstone projects.

## Repository Structure

```
DataScience_Training/
├── week-01-data-science/          # Foundations, Wrangling, Visualization, ML Basics, Regression, Classification, Unsupervised Learning, Cross-Validation
│   ├── scenario-based/            # Day-by-day exercises (Day 01 - Day 08)
│   └── capstone/                  # Capstone Projects 1 & 2
│
├── week-02-ai-ml/                 # Scikit Pipelines, Deep Learning, Advanced CV
│   ├── scenario-based/            # Day-by-day exercises (Day 09 - Day 11)
│   └── capstone/                  # Capstone Projects 3 & 4
│
├── week-03-ai-ml-advanced-prompt/ # Transformers, Prompt Engineering, Vector Embeddings
│   ├── scenario-based/            # Day-by-day exercises (Day 12 - Day 14)
│   └── capstone/                  # AI Research Assistant & Waste Classification
│
├── extra-curricular-activity/     # Quizzes, Datasets
│   ├── quizzes/
│   └── dataset/
│
├── requirements.txt
├── debugger.py
└── NOTES.md
```

## Weekly Overview

### Week 01 - Data Science Foundations (Feb 21 - Mar 2)
| Day | Topic | Key Concepts |
|-----|-------|-------------|
| 01 | Foundations | NumPy, Pandas, Arrays, DataFrames |
| 02 | Data Wrangling | Data Cleaning, Transformation, Database Scenarios |
| 03 | Visualization & Probability | Matplotlib, Seaborn, Probability Distributions |
| 04 | ML Basics | Train/Test Split, Feature Selection, Salary Prediction |
| 05 | Regression | Linear/Multiple Regression, EDA |
| 06 | Classification | Logistic Regression, KNN, BMI/Spam Detection |
| 07 | Unsupervised Learning | Clustering, DBSCAN |
| 08 | Cross-Validation | K-Fold CV, Bias-Variance Tradeoff |

### Week 02 - AI/ML (Mar 3 - Mar 9)
| Day | Topic | Key Concepts |
|-----|-------|-------------|
| 09 | Scikit Pipeline | Preprocessing Pipelines, Feature Engineering |
| 10 | Deep Learning | Perceptrons, Multi-layer Networks, Neural Networks |
| 11 | Advanced CV | Computer Vision, Image Classification, CNNs |

### Week 03 - Advanced AI/ML & Prompt Engineering (Mar 11 - Mar 14)
| Day | Topic | Key Concepts |
|-----|-------|-------------|
| 12 | Transformers | Attention Mechanism, BERT, GPT |
| 13 | Prompt Engineering | Prompt Design, LLM Interaction |
| 14 | Vector Embeddings | Vector Databases, Similarity Search, FAISS |
